En este directorio se encuentran las librerias firmadas para que los applet puedan acceder a ellas.
Para firmar librerias bajar el archivo build.xml y modificarlo para que tomes las librerias nuevas, especificando los path en donde se encuentran.
Se debe tener instalado el ant y el jdk
Estas librerias deben estar en la carmeta lib/, la cual se debe crear a la misma altura que WEB-INF de la parte WEB
